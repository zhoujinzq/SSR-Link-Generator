# SSR Link Generator


## 用途

按照所选设置，批量生成base64加密的SSR账号链接，方便机场地勤分发。

- 批量贴入密码，生成对应数量的账号链接
- 可直接粘贴回excel等文档，自动分行
- 支持保存本次使用值，便于后续修改
- 可自由编辑加密方式、混淆、协议菜单选项



## 系统要求

Mac OS 10.13 +

## 编译

XCode 9.4+
